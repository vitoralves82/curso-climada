Prática 1: Instalação do Ambiente CLIMADA e Primeiros Passos

Nesta prática, você irá configurar o ambiente CLIMADA em seu computador e executar seus primeiros comandos com a API da plataforma. Este é o ponto de partida para todas as análises de risco climático que realizaremos ao longo do curso.

Objetivos desta prática:

  • Instalar o ambiente Python necessário para CLIMADA
  • Configurar o CLIMADA utilizando conda/mamba
  • Verificar a instalação com testes simples
  • Executar comandos básicos utilizando a API do CLIMADA

1. Requisitos do Sistema

Antes de começarmos a instalação, verifique se seu sistema atende aos seguintes requisitos:

Hardware Recomendado:

  • Processador: Multi-core moderno (Intel i5/i7 ou equivalente)
  • Memória RAM: Mínimo 8GB, recomendado 16GB
  • Espaço em disco: Mínimo 10GB disponíveis

Software Necessário:

  • Sistema Operacional: Windows 10/11, macOS, ou Linux
  • Git instalado (para clonar repositórios)
  • Conexão com a Internet para download de pacotes

Dica profissional:

Para melhores resultados, recomendamos utilizar um ambiente dedicado para o CLIMADA, isolado de outras instalações Python. Isso evita conflitos de dependências com outros projetos.

2. Instalação do Ambiente CLIMADA

O CLIMADA é disponibilizado pelo conda-forge e é recomendado instalá-lo usando o gerenciador de pacotes Mamba (uma implementação mais rápida do Conda).

2.1 Instalação do Miniconda e Mamba

Primeiro, instale o Miniconda:

  1. Baixe o instalador do Miniconda apropriado para seu sistema operacional:
    https://docs.conda.io/en/latest/miniconda.html
  2. Execute o instalador e siga as instruções na tela
  3. Abra um novo terminal para verificar a instalação com o comando conda --version

Depois, instale o Mamba:

Terminal 
conda install -c conda-forge mamba

2.2 Criação do Ambiente CLIMADA

Agora vamos criar um ambiente dedicado para o CLIMADA e instalar o pacote:

Terminal 
# Crie um novo ambiente chamado 'climada_env'
mamba create -n climada_env -c conda-forge python=3.9

# Ative o ambiente
conda activate climada_env

# Instale o CLIMADA
mamba install -c conda-forge climada

# Verificação opcional: instale o Jupyter Notebook para trabalhar com tutoriais
mamba install -c conda-forge jupyterlab notebook

Atenção:

A instalação do CLIMADA pode demorar alguns minutos devido ao grande número de dependências. Se encontrar conflitos de pacotes, tente criar um ambiente totalmente novo com o comando acima. Não misture pip e conda para instalação de pacotes no mesmo ambiente.

3. Verificação da Instalação

Vamos verificar se o CLIMADA foi instalado corretamente executando alguns comandos básicos:

Python 
# Teste básico para verificar a instalação do CLIMADA
import sys
import climada
import numpy as np
import matplotlib.pyplot as plt

# Imprima as versões
print(f"Python version: {sys.version}")
print(f"CLIMADA version: {climada.__version__}")
print(f"NumPy version: {np.__version__}")

# Teste a importação de módulos essenciais do CLIMADA
from climada.hazard import Hazard
from climada.entity import Entity, Exposures
from climada.engine import Impact

print("Importações básicas realizadas com sucesso!")

Salve o código acima em um arquivo verificacao_climada.py e execute-o no terminal:

Terminal 
python verificacao_climada.py

Se não ocorrer nenhum erro e você vir a mensagem "Importações básicas realizadas com sucesso!", a instalação está correta.

4. Primeiros Passos com a API do CLIMADA

Agora vamos explorar os conceitos básicos da API do CLIMADA. Crie um notebook Jupyter ou um script Python e execute os seguintes exemplos:

4.1 Criando uma grade de centroides

Centroides são pontos geográficos onde o CLIMADA calcula a intensidade dos perigos e impactos. Vamos criar e visualizar uma grade simples:

Python 
import numpy as np
import matplotlib.pyplot as plt
from climada.hazard import Centroids

# Definir uma área para análise (exemplo: parte da costa brasileira)
min_lat, max_lat = -25.0, -22.0  # latitude sul (valores negativos)
min_lon, max_lon = -45.0, -42.0  # longitude oeste (valores negativos)

# Criar centroides com resolução de 0.1 graus
centroides = Centroids.from_pnt_bounds(
    (min_lon, min_lat, max_lon, max_lat), 
    res=0.1
)

# Verificar os centroides
centroides.check()

# Plotar os centroides
plt.figure(figsize=(10, 8))
centroides.plot()
plt.title('Grade de Centroides - Costa Sudeste do Brasil')
plt.xlabel('Longitude')
plt.ylabel('Latitude')
plt.show()

4.2 Explorando exposições econômicas com LitPop

O módulo LitPop permite estimar a distribuição espacial de valores econômicos. Vamos criar uma exposição simples:

Python 
from climada.entity.exposures import LitPop

# Criar uma exposição LitPop para uma região específica
# Usando um país pequeno para processamento mais rápido
exposicao = LitPop.from_countries(
    'URY',  # Uruguai (código ISO)
    res_arcsec=120,  # Resolução em segundos de arco (2 minutos ~ 3.7 km no equador)
    reference_year=2020  # Ano de referência
)

# Definir geometria de pontos para visualização
exposicao.set_geometry_points()

# Visualizar a exposição
plt.figure(figsize=(12, 8))
exposicao.plot_hexbin(
    pop_name=True,  # Mostrar nome do país
    buffer=0.5,     # Buffer ao redor da área
    cmap='YlOrRd'   # Esquema de cores
)
plt.title('Distribuição da Exposição Econômica - Uruguai')
plt.show()

# Mostrar estatísticas básicas
valor_total = exposicao.gdf.value.sum()
print(f"Valor total da exposição: {valor_total:,.2f} USD")
print(f"Número de pontos de exposição: {len(exposicao.gdf)}")
print(f"Valor médio por ponto: {exposicao.gdf.value.mean():,.2f} USD")
print(f"Valor máximo por ponto: {exposicao.gdf.value.max():,.2f} USD")

Dica:

Para usar o LitPop com dados demográficos, você precisa baixar manualmente o conjunto de dados Gridded Population of the World (GPW) do site da NASA SEDAC. Veja as instruções detalhadas na documentação oficial do LitPop.

Solução de Problemas Comuns

Erro ao baixar dados GPW para o LitPop

Para usar o LitPop, você precisa baixar manualmente o conjunto de dados GPW. Siga as instruções em: Tutorial LitPop - Population Count

Conflitos de pacotes durante a instalação

Se encontrar conflitos, tente criar um ambiente totalmente novo e siga a ordem exata de instalação. Evite misturar pip e conda no mesmo ambiente.

Problemas de memória com grandes áreas geográficas

Ao trabalhar com grandes áreas geográficas, reduza a resolução dos centroides ou divida a análise em regiões menores para evitar problemas de memória.
