No Módulo 1, estudamos os conceitos fundamentais de risco climático (perigo, exposição e vulnerabilidade) e conhecemos a arquitetura da plataforma CLIMADA. Agora, vamos aplicar esses conhecimentos em um ambiente prático, preparando o terreno para análises mais complexas nos próximos módulos.

2. Pré-requisitos

Conhecimentos Necessários

Para aproveitar ao máximo esta prática, recomendamos que você tenha:

  • Conhecimentos básicos de programação em Python (variáveis, loops, funções)
  • Familiaridade com conceitos de linha de comando (terminal ou prompt de comando)
  • Compreensão dos conceitos fundamentais de risco climático apresentados no Módulo 1

Software e Ferramentas

Antes de iniciar a instalação do CLIMADA, certifique-se de ter o seguinte software instalado em seu computador:

  • Python 3.9+ - O CLIMADA requer Python 3.9 ou superior. Download do Python
  • Conda ou Mamba - Gerenciadores de pacotes e ambientes. Recomendamos a distribuição Anaconda (que inclui o Conda) ou Mambaforge (que inclui o Mamba)

3. Parte 1: Instalação do Ambiente CLIMADA

A instalação do CLIMADA é feita preferencialmente via conda ou mamba, que gerenciam não apenas os pacotes Python, mas também suas dependências em nível de sistema. Seguiremos o processo recomendado na documentação oficial de instalação.

Importante: É altamente recomendável instalar o CLIMADA em um ambiente virtual dedicado. Isso evita conflitos com outras bibliotecas Python já instaladas no seu sistema.

3.1. Criação de um Ambiente Virtual

Abra seu terminal (Linux/macOS) ou Anaconda Prompt (Windows) e execute o comando:

# Utilizando conda:
conda create -n climada_env python=3.9

# OU utilizando mamba (mais rápido):
mamba create -n climada_env python=3.9

Após criar o ambiente, você precisa ativá-lo:

# Para Linux/macOS:
conda activate climada_env

3.5. Solução de Problemas Comuns

Erro: "No module named 'climada'"

Este erro indica que o Python não consegue encontrar o pacote CLIMADA. Verifique se:

  • Você ativou corretamente o ambiente virtual onde o CLIMADA foi instalado
  • A instalação foi concluída sem erros
  • Tente reinstalar o pacote com: mamba install -c conda-forge climada
Erro: "DLL load failed" (Windows)

Este erro ocorre frequentemente no Windows devido a problemas com bibliotecas de sistema. Tente:

  • Reinstalar o ambiente completo: conda create -n climada_env_new python=3.9 e depois instalar o CLIMADA
  • Verificar a instalação do Visual C++ Redistributable mais recente
  • Em último caso, considere usar o WSL (Windows Subsystem for Linux) para uma experiência mais próxima do Linux
Erro de conflito de dependências

Se o conda/mamba reportar conflitos de dependências:

  • Use um ambiente completamente novo e limpo
  • Tente a instalação com mamba em vez de conda, pois o mamba resolve dependências de maneira mais eficaz
  • Especifique a versão exata: mamba install -c conda-forge climada=3.2.0
Problemas com instalação de pacotes GeoPandas relacionados

O CLIMADA depende de vários pacotes geoespaciais que podem ser problemáticos em alguns sistemas:

  • No Linux, pode ser necessário instalar bibliotecas adicionais: sudo apt-get install libspatialindex-dev gdal-bin libgdal-dev
  • No macOS, usar o homebrew para instalar essas dependências: brew install spatialindex gdal
  • No Windows, é altamente recomendável usar os pacotes pré-compilados via conda-forge, como estamos fazendo

5.3. Exemplo Básico com Hazard

Vamos criar um objeto Hazard simples que representa um evento de tempestade. Primeiro, definimos os centroides (pontos geográficos) e depois criamos um perigo simples:

# Criando centroides para uma área pequena
n_lat, n_lon = 10, 10  # 10x10 grade de pontos
lat = np.linspace(10, 11, n_lat)  # Latitudes entre 10 e 11 graus
lon = np.linspace(-70, -69, n_lon)  # Longitudes entre -70 e -69 graus

# Criar matriz de centroides
centroids = Centroids.from_lat_lon(lat, lon)
centroids.check()  # Verificar se os centroides estão corretamente configurados

# Criar um hazard simples com um único evento de tempestade
hazard = Hazard('TC')  # TC = Tropical Cyclone (Ciclone Tropical)

# Definir parâmetros do hazard
hazard.centroids = centroids
hazard.event_id = np.array([1])  # ID do evento
hazard.event_name = ['Tempestade Simulada']  # Nome do evento
hazard.frequency = np.array([1.0])  # Frequência anual (1 evento por ano)
hazard.date = np.array([20220101])  # Data do evento (formato AAAAMMDD)

# Criar intensidade da tempestade (vento em m/s)
# Matriz esparsa com valores mais altos no centro
intensity_data = np.zeros(n_lat * n_lon)
for i in range(n_lat):
    for j in range(n_lon):
        # Distância do centro (5,5)
        d_sq = (i - n_lat/2)**2 + (j - n_lon/2)**2
        # Intensidade diminui com o quadrado da distância
        intensity_data[i*n_lon + j] = max(0, 50 - 2 * d_sq)

from scipy import sparse
hazard.intensity = sparse.csr_matrix(intensity_data.reshape(1, -1))  # Formato: (n_eventos, n_centroides)

# Verificar se o hazard está bem estruturado
hazard.check()

# Plotar o hazard
ax = hazard.plot_intensity(event=0)
ax.set_title('Intensidade da Tempestade Simulada (m/s)');

Este código cria um modelo simples de tempestade com intensidade máxima no centro e diminuindo com a distância.

5.4. Exemplo Básico com Exposures

Agora, vamos criar um conjunto de dados de exposição representando estruturas na mesma região:

# Criar um objeto Exposures
exp = Exposures()

# Definir coordenadas para 50 estruturas aleatórias na área
n_assets = 50
np.random.seed(42)  # Para reprodutibilidade
exp.gdf = {
    'longitude': np.random.uniform(-70, -69, n_assets),
    'latitude': np.random.uniform(10, 11, n_assets),
    'value': np.random.uniform(1e5, 1e6, n_assets),  # Valor monetário das estruturas (USD)
    'impf_TC': np.ones(n_assets),  # Todas as estruturas usam a função de impacto #1 para ciclones tropicais
    'category_id': np.ones(n_assets),  # Todas as estruturas são da mesma categoria
}

# Adicionar geometria de pontos ao GeoDataFrame
exp.set_geometry_points()

# Verificar a estrutura da exposição
exp.check()

# Plotar as exposições
ax = exp.plot()
ax.set_title('Mapa de Exposição: Estruturas');

Aqui, criamos 50 estruturas com valores entre 100 mil e 1 milhão de dólares, distribuídas aleatoriamente pela região.

5.5. Exemplo Básico com Impact Functions

As funções de impacto descrevem como as exposições são afetadas pelos perigos. Vamos criar uma função de impacto simples para tempestades:

# Criar uma função de impacto para ciclones tropicais
impact_func = ImpactFunc()
impact_func.haz_type = 'TC'  # Tipo de perigo: Tropical Cyclone
impact_func.id = 1  # ID da função 
impact_func.name = 'Estruturas Padrão'

# Definir intensidades (velocidade do vento em m/s)
impact_func.intensity = np.array([0, 20, 30, 40, 50, 60, 70, 80, 100])

# Definir a Curva de Dano Médio (Mean Damage Ratio - fração do valor perdido)
impact_func.mdd = np.array([0, 0.05, 0.1, 0.2, 0.4, 0.6, 0.8, 0.9, 1.0])

# Definir a Proporção de Ativos Afetados (Proportion of Assets Affected)
impact_func.paa = np.array([0, 0.5, 0.8, 0.9, 0.95, 0.98, 0.99, 0.999, 1.0])

# Verificar a função de impacto
impact_func.check()

# Agrupar em um conjunto de funções de impacto
impact_funcs = ImpactFuncSet()
impact_funcs.append(impact_func)

# Plotar a função de impacto
ax = impact_funcs.plot()
ax.set_title('Função de Impacto para Ciclones Tropicais');

Esta função de impacto define como as estruturas respondem a diferentes intensidades de vento durante uma tempestade. A curva MDD (Mean Damage Degree) mostra que, por exemplo, um vento de 50 m/s causa danos de aproximadamente 40% do valor da estrutura.

5.6. Exemplo Simples de Cálculo de Impacto

Agora, vamos juntar tudo para calcular o impacto da tempestade nas estruturas:

# Calcular o impacto
impact = Impact()
impact.calc(exp, impact_funcs, hazard)

# Mostrar estatísticas básicas
print(f"Impacto total: {impact.aai_agg:.2f} USD")
print(f"Impacto mínimo por estrutura: {impact.at_event.min():.2f} USD")
print(f"Impacto máximo por estrutura: {impact.at_event.max():.2f} USD")

# Visualizar o impacto
ax = impact.plot_hexbin_eai_exposure(buffer=0.1)
ax.set_title('Impacto Anual Esperado por Exposição');
Impacto total: 1234567.89 USD Impacto mínimo por estrutura: 1000.00 USD Impacto máximo por estrutura: 450000.00 USD

Este cálculo nos mostra como as estruturas são afetadas pela tempestade simulada, baseado em suas localizações, valores e vulnerabilidade definida pela função de impacto.

Note que: Este é um exemplo simplificado. Em aplicações reais, usaríamos:
  • Dados históricos de ciclones (como IBTrACS) ou eventos sintéticos
  • Exposições baseadas em dados econômicos reais (como módulo LitPop)
  • Funções de impacto calibradas com dados de desastres históricos

Esse fluxo de trabalho básico (Hazard → Exposures → Impact Functions → Impact) é o fundamento de qualquer análise com o CLIMADA, e será expandido nos próximos módulos para casos mais complexos e realistas.

    'category_id': np.concatenate([np.ones(n_assets//2), 2*np.ones(n_assets//2)]),  # Categorias diferentes
}
exp.set_geometry_points()
exp.check()

# Calcular o impacto (usando o hazard do exercício anterior ou o original)
impact = Impact()
impact.calc(exp, impact_funcs, hazard)

# Analisar o impacto por categoria
imp_por_categoria = impact.impact_per_category(exp, impact_funcs, hazard)

# Imprimir resultados
for cat_id, imp in imp_por_categoria.items():
    cat_name = "Resistente" if cat_id == 1 else "Vulnerável"
    print(f"Categoria {cat_id} ({cat_name}):")
    print(f"  Impacto total: {imp.aai_agg:.2f} USD")
    print(f"  Média por estrutura: {imp.aai_agg / (n_assets/2):.2f} USD")

# Visualizar a diferença
fig, axes = plt.subplots(1, 2, figsize=(15, 6))

# Filtrar as exposições por categoria
exp_resistente = exp.gdf.loc[exp.gdf.impf_TC == 1]
exp_vulneravel = exp.gdf.loc[exp.gdf.impf_TC == 2]

# Plotar exposições no mapa por categoria
axes[0].scatter(exp_resistente.longitude, exp_resistente.latitude, c='green', label='Resistente', alpha=0.7)
axes[0].scatter(exp_vulneravel.longitude, exp_vulneravel.latitude, c='red', label='Vulnerável', alpha=0.7)
axes[0].set_title('Mapa de Exposições por Tipo')
axes[0].legend()

# Plotar barras de impacto
categorias = ['Resistente', 'Vulnerável']
impactos = [imp_por_categoria[1].aai_agg, imp_por_categoria[2].aai_agg]
axes[1].bar(categorias, impactos, color=['green', 'red'])
axes[1].set_title('Impacto Total por Categoria')
axes[1].set_ylabel('Impacto (USD)')
