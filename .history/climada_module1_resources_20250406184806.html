1. Introdução

Objetivo da Página de Recursos

Esta página de recursos complementares foi desenvolvida para enriquecer sua jornada de aprendizado no Módulo 1 do curso "Precificação de Riscos Climáticos em Ativos Corporativos com CLIMADA". Aqui você encontrará materiais adicionais cuidadosamente selecionados que aprofundam os conceitos apresentados no módulo, oferecendo perspectivas diversas e aplicações práticas.

Os recursos estão organizados em categorias temáticas e classificados por nível de complexidade (iniciante, intermediário e avançado), permitindo que você construa seu conhecimento progressivamente, de acordo com seu ritmo e interesses específicos.

Como Utilizar os Materiais Complementares

Para aproveitar ao máximo esta página de recursos:

  • Explore por categorias: Navegue pelas seções temáticas que correspondem aos seus interesses específicos ou às áreas que deseja aprofundar.
  • Utilize os filtros: Use a barra de pesquisa e os filtros por nível para encontrar rapidamente os recursos mais relevantes para você.
  • Siga a progressão recomendada: Para cada unidade do Módulo 1, sugerimos uma sequência de recursos que complementa o conteúdo do curso de forma estruturada.
  • Crie seu percurso personalizado: Adapte seu aprendizado combinando diferentes recursos de acordo com seus objetivos profissionais e áreas de interesse.

Recomendação: Reserve um tempo específico para explorar estes recursos após completar cada unidade do Módulo 1. Esta prática ajudará a consolidar seu aprendizado e expandir sua compreensão dos conceitos fundamentais.

2. Artigos Científicos e Publicações

CLIMADA v1: a global weather and climate risk assessment platform

Iniciante

Aznar-Siguan, G. and Bresch, D. N. (2019)

Este artigo fundamental apresenta a plataforma CLIMADA, sua arquitetura e funcionalidades principais. É uma excelente introdução aos conceitos fundamentais e à estrutura do modelo que usaremos ao longo do curso.

Acessar artigo
Fundamentos Arquitetura

A framework for global river flood risk assessments

Intermediário

Winsemius, H. C., et al. (2013)

Uma abordagem metodológica para avaliação de risco de inundações em escala global. Este artigo demonstra conceitos fundamentais de modelagem de perigos climáticos que são aplicados no CLIMADA.

Acessar artigo
Perigos Modelagem

Uncertainty and sensitivity analysis for probabilistic weather and climate risk modelling

Avançado

Eberenz, S., et al. (2022)

Este artigo discute a implementação de métodos de análise de incerteza e sensibilidade no CLIMADA, essenciais para entender as limitações da modelagem de risco climático e para comunicar os resultados de forma transparente.

Acessar artigo
Incerteza Modelagem

Asset exposure data for global physical risk assessment

Intermediário

Eberenz, S., et al. (2020)

Explica o desenvolvimento do LitPop, um método utilizado no CLIMADA para estimar a exposição econômica global. Fundamental para entender como são modelados os valores de ativos expostos a riscos climáticos.

Acessar artigo
Exposição LitPop

The impacts of climate hazards on offshore energy infrastructure: A review

Intermediário

Marshall, K. M., et al. (2021)

Uma revisão abrangente sobre os impactos dos perigos climáticos em infraestruturas energéticas offshore. Especialmente relevante para o foco do curso em embarcações e estruturas offshore.

Acessar artigo
Offshore Vulnerabilidade

5. Ferramentas e Datasets Complementares

IBTrACS - International Best Track Archive for Climate Stewardship

Iniciante

Dataset global de trajetórias de ciclones tropicais, utilizado pelo CLIMADA para modelagem de ciclones. Contém registros históricos detalhados desde 1850 até o presente.

Acessar dataset
Dados Ciclones

CMIP6 - Coupled Model Intercomparison Project Phase 6

Intermediário

Conjunto de simulações de modelos climáticos globais que fornece projeções futuras para diversos cenários de emissões. Útil para análise de riscos climáticos a longo prazo.

Acessar dataset
Dados Projeções

API de Dados CLIMADA

Iniciante

Interface para acessar dados pré-processados utilizados no CLIMADA, incluindo perigos, exposições e funções de impacto. Facilita o acesso a dados sem necessidade de processamento local.

Acessar API
API Serviço

ERA5 - Climate Reanalysis

Intermediário

Dataset de reanálise climática produzido pelo ECMWF, fornecendo dados históricos de alta resolução para diversas variáveis climáticas. Importante para análise de eventos extremos passados.

Acessar dataset
Dados Reanálise

Cartopy - Biblioteca de Mapeamento Geoespacial

Iniciante

Biblioteca Python para criação de mapas e visualizações geoespaciais, utilizada pelo CLIMADA para representação de resultados. Útil para personalização de visualizações além do padrão do CLIMADA.
