Perguntas Frequentes

Como instalar o CLIMADA em um ambiente virtual?

Para instalar o CLIMADA em um ambiente virtual, siga estes passos:

  1. Instale o Anaconda ou Miniconda
  2. Crie um novo ambiente com: conda create -n climada_env python=3.9
  3. Ative o ambiente: conda activate climada_env
  4. Instale o CLIMADA: conda install -c conda-forge climada
  5. Verifique a instalação executando um script de teste

Para mais detalhes, consulte o guia de instalação completo.

Como baixar os dados do GPW para análises de população?

Os dados do Gridded Population of the World (GPW) precisam ser baixados manualmente:

  1. Acesse o site SEDAC GPW
  2. Crie uma conta gratuita (se ainda não tiver)
  3. Baixe os arquivos de contagem populacional (Population Count) no formato GeoTIFF
  4. Coloque os arquivos na pasta de sistema do CLIMADA (geralmente em ~/climada/data/)

Para verificar o caminho correto da pasta, consulte a documentação ou execute import climada; print(climada.CONFIG.data_dir)

Como resolver erros comuns de memória ao processar grandes conjuntos de dados?

Erros de memória são comuns ao processar grandes conjuntos de dados no CLIMADA. Algumas soluções incluem:

  • Reduzir a resolução espacial usando parâmetros como res_arcsec com valores maiores
  • Limitar a análise a uma região geográfica menor usando from_shape em vez de países inteiros
  • Usar multiprocessamento com pool para processamento paralelo
  • Ajustar o parâmetro CONFIG.max_matrix_size para controlar o uso de memória

Se continuar enfrentando problemas, considere usar uma máquina com mais RAM ou plataformas de cloud computing.

Como criar funções de impacto customizadas para ativos específicos?

Para criar uma função de impacto customizada para um tipo específico de ativo:

  1. Crie uma instância de ImpactFunc
  2. Defina o ID da função e o tipo de perigo
  3. Configure os arrays de intensidade, MDD (Mean Damage Degree) e PAA (Percentage of Assets Affected)
  4. Adicione a função a um ImpactFuncSet
import numpy as np
from climada.entity import ImpactFunc, ImpactFuncSet

# Criar função de impacto
func = ImpactFunc()
func.id = 1
func.haz_type = 'TC'
func.name = 'Embarcação Offshore'
func.intensity_unit = 'm/s'
func.intensity = np.array([0, 20, 30, 40, 50, 60, 70, 80, 100])
func.mdd = np.array([0, 0, 0.1, 0.2, 0.4, 0.6, 0.8, 0.9, 1.0])
func.paa = np.array([0, 0.1, 0.2, 0.4, 0.6, 0.8, 0.9, 0.95, 1.0])

# Adicionar a um conjunto de funções
impact_funcs = ImpactFuncSet()
impact_funcs.append(func)
Como interpretar as métricas de risco do CLIMADA?

As principais métricas de risco do CLIMADA incluem:

  • Average Annual Impact (AAI): O impacto médio anual esperado, considerando todos os eventos na base de dados e suas frequências.
  • Probable Maximum Loss (PML): A perda máxima provável para diferentes períodos de retorno (ex: 10, 100, 250 anos).
  • Exceedance Probability Curve (EPC): Mostra a probabilidade de exceder um determinado nível de impacto.

Para acessar estas métricas após calcular um impacto:

# AAI (Average Annual Impact)
print(f"AAI: {imp.aai_agg}")

# Calcular e plotar a curva de frequência/probabilidade
freq_curve = imp.calc_freq_curve()
freq_curve.plot()

# PML para diferentes períodos de retorno
return_periods = [10, 25, 50, 100, 250]
pml = imp.calc_pml(return_periods)
print(pml)
