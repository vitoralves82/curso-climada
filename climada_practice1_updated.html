Prática 1: Instalação e Primeiros Passos

Configure seu ambiente, realize testes iniciais com CLIMADA e explore a API básica. Ao final, você terá um ambiente funcional e familiaridade com os componentes essenciais.

Conteúdo da Prática

1. Instalação do Ambiente CLIMADA 2. Configuração do Jupyter Notebook/Lab 3. Primeiro Contato com a API CLIMADA 4. Exercícios Práticos

Pré-requisitos

  • Conhecimentos básicos de linha de comando (terminal/prompt)
  • Familiaridade com Python (variáveis, funções, importações)
  • Computador com acesso à internet e 8GB+ RAM (16GB+ recomendado)
  • Instalador Mamba ou Conda (Miniconda/Anaconda)

1. Instalação do Ambiente CLIMADA

Instalaremos o CLIMADA usando Mamba (recomendado) em um ambiente virtual dedicado.

1.1. Criação do Ambiente Virtual

Abra seu terminal e crie o ambiente: 

mamba create -n climada_env python=3.9 -c conda-forge -y

# O '-y' confirma automaticamente.

1.2. Ativação do Ambiente

Ative o ambiente: 

mamba activate climada_env

# Seu prompt deve indicar "(climada_env)".

1.3. Instalação do CLIMADA Core

Instale o pacote principal: 

mamba install -c conda-forge climada -y

Atenção: Pode demorar. Se houver erros, tente: mamba install -c conda-forge climada --strict-channel-priority -y

1.4. Verificação da Instalação

Verifique no terminal: 

python -c "import climada; print(f'CLIMADA versão: {climada.__version__}')"

A versão instalada deve ser exibida.

Opcional: CLIMADA Petals

Para funcionalidades extras, instale:

mamba install -c conda-forge climada_petals -y

2. Configuração do Jupyter Notebook/Lab

Usaremos JupyterLab para análises interativas.

2.1. Instalação do JupyterLab

No ambiente climada_env ativo, instale: 

mamba install -c conda-forge jupyterlab -y

2.2. Iniciando o JupyterLab

Navegue até sua pasta de trabalho e inicie: 

jupyter lab

Seu navegador abrirá a interface do JupyterLab.

2.3. Criando seu Notebook

No "Launcher", clique em "Python 3 (ipykernel)". Renomeie para Pratica1_CLIMADA.ipynb.

2.4. Teste Rápido no Notebook

Na primeira célula, execute (Shift+Enter): 

import climada
print(f"CLIMADA versão: {climada.__version__}")

A versão deve ser exibida, confirmando a configuração.

Dica: Atalhos úteis: Shift+Enter (executa e avança), Ctrl+Enter (executa no local), Esc (modo de comando), M (mudar para Markdown), Y (mudar para código).

3. Primeiro Contato com a API CLIMADA

Explore os componentes básicos no seu notebook Pratica1_CLIMADA.ipynb.

3.1. Importações Essenciais

Comece importando os módulos necessários: 

import numpy as np
import matplotlib.pyplot as plt
from pathlib import Path # Para gerenciar caminhos

# Componentes CLIMADA
from climada.hazard import Hazard, Centroids, TCTracks, TropCyclone
from climada.entity import Exposures, Entity, ImpactFuncSet
from climada.entity.impact_funcs.trop_cyclone import ImpfTropCyclone
from climada.engine import Impact

# Configuração para gráficos inline
%matplotlib inline
# Define diretório de dados
DATA_DIR = Path.home() / "climada_data"
DATA_DIR.mkdir(parents=True, exist_ok=True)
print(f"Diretório de dados: {DATA_DIR}")

3.2. Centroids: Pontos de Cálculo

Crie centroids para as Filipinas Centrais: 

lat_min, lat_max = 8, 15
lon_min, lon_max = 120, 127
resolution_deg = 0.1

centroids_phl = Centroids.from_pnt_bounds((lon_min, lat_min, lon_max, lat_max), res=resolution_deg)
centroids_phl.check()

fig, ax = plt.subplots(1, 1, figsize=(8, 8))
centroids_phl.plot(ax=ax, marker='.', s=1) # Marker menor
ax.set_title(f"{len(centroids_phl.lat)} Centroids nas Filipinas Centrais")
plt.show()

Representam locais onde o perigo e o impacto serão calculados.

3.3. TCTracks: Trajetórias de Ciclones

Carregue dados do Tufão Haiyan (2013): 

try:
    tracks_haiyan = TCTracks.from_ibtracs_netcdf(storm_id='2013306N06150', provider='usa', data_dir=DATA_DIR / "ibtracs")
    tracks_haiyan.equal_timestep(time_step_h=1.0)

    fig, ax = plt.subplots(1, 1, figsize=(10, 6))
    tracks_haiyan.plot(ax=ax)
    centroids_phl.plot(ax=ax, color='gray', alpha=0.3, markersize=1) # Mais transparente
    ax.set_title(f"Trajetória - {tracks_haiyan.data[0].attrs['name']} ({tracks_haiyan.data[0].attrs['sid']})")
    plt.show()
    # print(tracks_haiyan.data[0]) # Descomente para ver detalhes
except Exception as e:
    print(f"Erro IBTrACS: {e}. Verifique conexão/diretório.")

Download: Pode demorar na primeira vez. Necessita internet.

3.4. TropCyclone (Hazard): Modelando o Perigo

Modele o campo de vento nos centroids: 

tc_haiyan_hazard = TropCyclone.from_tracks(tracks_haiyan, centroids=centroids_phl)
tc_haiyan_hazard.check()

fig, ax = plt.subplots(1, 1, figsize=(10, 8))
tc_haiyan_hazard.plot_intensity(event=0, ax=ax)
ax.set_title(f"Intensidade Máx. Vento (m/s) - {tracks_haiyan.data[0].attrs['name']}")
plt.show()

Contém a intensidade do perigo (vento) em cada centroid.

3.5. Exposures: Modelando Ativos

Crie exposição econômica com LitPop: 

from climada.entity.exposures.litpop import LitPop

try:
    exp_phl = LitPop.from_countries('PHL', res_arcsec=300, data_dir=DATA_DIR / "litpop")
    exp_phl.check()
    exp_phl.gdf['impf_TC'] = 1 # Associar função de impacto ID 1
    exp_phl.crop(centroids_phl.buffer) # Opcional: filtrar para área de centroids

    fig, ax = plt.subplots(1, 1, figsize=(10, 8))
    exp_phl.plot_hexbin(ax=ax, pop_name=False, buffer=0.2, vmin=1e4) # Ajustar escala de cor
    ax.set_title(f"Exposição LitPop (USD) - Filipinas Centrais")
    plt.show()
    print(f"Exposição: {len(exp_phl.gdf)} pontos, Valor Total: {exp_phl.gdf.value.sum():,.0f} USD")
except Exception as e:
    print(f"Erro LitPop: {e}. Pode precisar baixar dados (GPW/Nightlight).")

Contém localização e valor dos ativos expostos.

3.6. Impact Functions: Modelando Vulnerabilidade

Defina a relação intensidade-dano: 

impf_tc_phl = ImpfTropCyclone.from_emanuel_asia()
impf_tc_phl.id = 1 # ID usado na exposição
impf_tc_phl.check()
impact_func_set = ImpactFuncSet([impf_tc_phl])

fig, ax = plt.subplots(1, 1, figsize=(8, 5))
impact_func_set.plot(axis=ax)
ax.set_title("Função Impacto TC (Emanuel Ásia)")
ax.set_xlabel("Velocidade Vento (m/s)")
ax.set_ylabel("Fração Média Dano (MDR)")
plt.grid(True, linestyle='--', alpha=0.6) # Adicionar grade
plt.show()

Contém as curvas de vulnerabilidade.

3.7. Engine (Impact): Calculando o Impacto

Combine tudo para calcular o impacto financeiro: 

impact_haiyan = Impact()
impact_haiyan.calc(exp_phl, impact_func_set, tc_haiyan_hazard)
impact_haiyan.check()

total_impact_usd = impact_haiyan.imp_mat.sum()
print(f"Impacto total estimado (Haiyan): {total_impact_usd:,.0f} USD")

fig, ax = plt.subplots(1, 1, figsize=(10, 8))
impact_haiyan.plot_spatial_impact(event=0, haz_id=tc_haiyan_hazard.event_id[0], ax=ax, vmin=1e5) # Ajustar vmin
ax.set_title(f"Impacto Espacial Estimado - {tracks_haiyan.data[0].attrs['name']} (USD)")
plt.show()

Objeto Impact contém os danos calculados.

Parabéns!

Você executou sua primeira análise de impacto ponta-a-ponta!
